## Dependencies

```

[dependencies]
teloxide = "0.2.0"
futures = "0.3.4"
tokio = "0.2.13"
log = "0.4.8"
```

## Installation:

Create file named `.env` or rename `.env.sample` to `.env` and change the following

```

TELOXIDE_TOKEN=123456890:ABCDEFGHIKLMN-OPQRS

```
